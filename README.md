<h3 align="center">测试工具平台</h3><hr />

QtestTool是一个测试工具集合网站，能有效助力测试团队加快高质量软件的交付。

### QtestTool 的功能

- **测试思路转换(xmind2case)**: 将xmind格式的测试思路转换为测试案例写入表格中；
- **测试挡板(testFaker)**: 测试挡板功能；
  

### 技术栈

-   后端: [flask](https://flask.net.cn/)
-   中间件: [MySQL](https://www.mysql.com/)
